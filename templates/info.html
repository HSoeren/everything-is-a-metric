everything is a metric

Metrics are scraped and used by prometheus. Normally your device offers an endpoint which prometheus can scrape. If not prometheus allows to be pushed with updates. Unfortunatly my usecase involves some devices that go dark for 20 days, then push three updates and go dark again for a long time. These created issues with tracking certain stuff which sparked the idea of a simple GET/POST based collection service which can be scraped itself.

GitHub: HSoeren/everything-is-a-metric

Hint: This is not meant as public service, but it needs to be reachable from the outside in order to work. So if you find this please be so kind to not use this service and not spread the word of this instance. You can however share the github project if you like - or contribute to it. Thanks a lot
